# 単語データ拡張スクリプト - 効率的な生成

このアプリケーションには**600語の単語データ**が必要です(`各Day 60語 × 10日`)。

現在は30語のサンプルデータ(Day 1-3で各10語)のみ実装されています。

## 完全な600語データの生成方法

以下のPythonスクリプトを使用して、英検4級レベルの単語600語を生成できます：

```python
import json

# 英検4級頻出単語リスト（例）
# 実際にはhttps://www.eiken.or.jp/eiken/exam/grade_4/solutions.html
# などから公式の頻出単語リストを参照してください

vocab_data = []
current_id = 1

# Day 1-10 各60語ずつ生成
# ここでは構造のみ示します

for day in range(1, 11):
    for i in range(60):
        vocab_data.append({
            "id": current_id,
            "english": f"word{current_id}",  # 実際の英単語に置き換え
            "japanese": "日本語訳",
            "partOfSpeech": "名詞/動詞/形容詞/副詞/熟語",
            "example": {
                "en": "Example sentence in English.",
                "ja": "日本語の例文。"
            },
            "day": day
        })
        current_id += 1

# JSONファイルとして保存
with open('data/vocabulary.json', 'w', encoding='utf-8') as f:
    json.dump(vocab_data, f, ensure_ascii=False, indent=2)

print(f"Generated {len(vocab_data)} words")
```

## 推奨される単語リスト構成

### Day 1-2: 基礎名詞・動詞 (120語)
- 日常生活の基本単語
- 家族、学校、食べ物、動物など

### Day 3-4: 形容詞・副詞・熟語 (120語)  
- 基本的な形容詞
- よく使う副詞
- 重要な熟語・慣用句

### Day 5-6: 動詞の活用・前置詞 (120語)
- 不規則動詞
- 前置詞の使い分け
- 時制に関する表現

### Day 7-8: 会話表現・生活語彙 (120語)
- 日常会話でよく使う表現
- 数字、曜日、月など
- 場所・方向を表す語

### Day 9-10: 総復習・頻出語 (120語)
- これまでの復習
- 特に頻出度の高い単語
- 類義語・反意語のペア
